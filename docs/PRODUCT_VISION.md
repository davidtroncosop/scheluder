# Scheduler Pro — Resumen de Funciones

> **Centro de comando inteligente para el armado de horarios académicos**, diseñado para reemplazar Excel y reducir errores operativos mediante asistencia automática, reglas claras y decisiones guiadas.

---

## 1️⃣ Gestión Inteligente de Asignación

### ✔ Asignación Guiada de Clases (NRC)
- El sistema identifica qué NRC son más difíciles de ubicar
- Prioriza automáticamente los casos críticos
- El usuario asigna con un click, sin arrastrar ni probar a ciegas

**Beneficio:** menos tiempo, menos prueba–error.

### ✔ Modo "Decisión Activa"
Al seleccionar un NRC:
- El sistema entra en modo foco
- Se muestran solo los slots válidos
- Se destacan las mejores opciones

**Beneficio:** el usuario piensa en una cosa a la vez.

---

## 2️⃣ Evaluación Automática de Disponibilidad

### ✔ Validación en Tiempo Real de Recursos
Cada decisión considera automáticamente:
- Disponibilidad de docentes
- Disponibilidad y tipo de salas
- Conflictos de nivel (alumnos)
- Reglas institucionales (RLS)

**Beneficio:** se evita crear errores sin saberlo.

### ✔ Scoring de Idoneidad de Slots
- Cada slot posible recibe un puntaje de calidad
- El sistema distingue entre:
  - ✅ Opciones óptimas
  - ⚠️ Aceptables
  - ❌ Prohibidas

**Beneficio:** el sistema recomienda, no impone.

---

## 3️⃣ Detección y Resolución de Conflictos

### ✔ Detección Automática de Conflictos
El planificador identifica en tiempo real:
- Duplicidad de docentes
- Choques de nivel
- Sobreuso de salas
- Incumplimiento de reglas críticas

**Beneficio:** los errores se detectan antes de publicar.

### ✔ Copiloto de Conflictos (Action Hub)
- Cada conflicto se presenta como una acción
- El usuario puede:
  - Entender el problema ("Ver por qué")
  - Resolverlo automáticamente

**Beneficio:** los errores no bloquean, se solucionan.

---

## 4️⃣ Vistas Dinámicas del Horario

### ✔ Cambio de Perspectiva
El mismo horario puede verse:
- 🏫 Por salas
- 📚 Por niveles/semestres
- 👨‍🏫 Por docentes

**Beneficio:** distintos roles ven lo que les importa.

### ✔ Modo Fantasma (Visualización Cruzada)
Permite ver de forma contextual:
- Carga de un docente
- Distribución de un nivel
- Ocupación de salas

**Beneficio:** se evitan choques invisibles.

---

## 5️⃣ Control y Salud del Sistema

### ✔ Indicador de Salud Global
Muestra en tiempo real:
- Porcentaje de NRC asignados
- Cantidad de conflictos críticos
- Alertas pendientes

**Beneficio:** el usuario sabe cuándo el horario está listo.

### ✔ Publicación Controlada
- El horario solo puede publicarse si no hay conflictos críticos
- El sistema explica por qué está bloqueado

**Beneficio:** se eliminan publicaciones erróneas.

---

## 6️⃣ Automatización y Asistencia Avanzada

### ✔ Resolución Automática de Asignaciones
- El sistema puede sugerir o aplicar la mejor ubicación para un NRC
- Usa reglas y scoring explicable

**Beneficio:** desbloquea casos complejos rápidamente.

### ✔ Reglas Explicables y Auditables
- Cada decisión puede justificarse
- Las reglas violadas son visibles

**Beneficio:** confianza institucional y trazabilidad.

---

## 7️⃣ Experiencia de Usuario y Productividad

### ✔ Reducción de Fricción
- No se arrastran bloques innecesariamente
- No se prueban combinaciones al azar
- No se necesita revisar todo manualmente

### ✔ Feedback Inmediato y Visual
Cada acción tiene respuesta clara:
- Visual
- Contextual
- No intrusiva

---

## 8️⃣ Escalabilidad y Evolución (Producto)

### ✔ Motor de Reglas Desacoplado
Permite:
- Agregar nuevas reglas
- Cambiar ponderaciones
- Versionar comportamiento

### ✔ Adaptable a Distintas Carreras e Instituciones
- No depende de Kinesiología
- Soporta múltiples contextos académicos

---

## 🏁 Propuesta de Valor

> **Scheduler Pro transforma el armado de horarios desde una tarea operativa caótica en un proceso guiado, confiable y estratégicamente controlado.**
